# 创建和管理测试轮廓

在Adobe Campaign中创建营销活动时，您将能够在发送的消息中创建动态内容和个性化。 测试用户档案允许您仔细检查您发送的消息是否按预期向用户显示。

了解如何在客户端控制台中创建测试配置文件，以及如何在Adobe Campaign Web UI中管理和编辑配置文件。
