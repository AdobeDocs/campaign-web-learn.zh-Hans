# 创建和管理测试轮廓

在 Adobe Campaign 中创建营销活动时，您可以在发送的消息中创建动态内容并进行个性化。测试轮廓可以帮助您再次确认所发送的消息是否会按预期呈现给用户。

了解如何在客户端控制台中创建测试轮廓，以及如何在 Adobe Campaign Web 用户界面中管理和编辑轮廓。
